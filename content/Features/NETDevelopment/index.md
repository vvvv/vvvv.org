vvvv is a .NET development environment in a sense comparable to [Visual Studio](https://visualstudio.microsoft.com/) and [Raider](https://www.jetbrains.com/rider/). What they share is their capability to develop apps using libraries that are available in the .NET ecosystem,

Only, vvvv has a focus on [visual live-programming](../VisualLiveProgramming/index.md) which benefits applications that require [rapid prototyping](../RapidPrototyping/index.md) or have IO that goes beyond the standard mouse/keayboard/touch-screen.

## Visual and textual code

While vvvv is a visual-first development environment, you can also choose to write parts of your apps using C#. vvvv's visual language VL is best at high/mid-level organization of code while text-code often excells at implementing fine-grained algorithms. The key is finding the right balance for yourself, which vvvv allows you to do. 

## Runtime value inspection

To inspect (and even tweak) the value of any variable in your app you don't need to pause the program, simply hover the element of interest with the mouse to see its value.
